Memory Game

by: Leighton Hsieh

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I thought it would be relatively easy to make. When I thought about the project and broke it down into its parts, they were seemingly simple and easy to make.

2. What I Learned

I learned I should be more accommodating to the user experience and should constantly ask myself "How is this gonna affect their experience playing the game?"

3. Architecture of My Game

The first thing you see is the intro/level select level with three different buttons. Pressing those buttons take you to a certain level. Each level has a grid of buttons and a start button which starts a timer when pressed. Every level is more challenging than the previous one. Level 1 has a 5x5 grid of buttons, level 2 has the same grid with a shorter amount of time given, and level 3 has a larger 7x7 grid. Completing a level takes you back to the intro level, with that button now removed. In order to complete the game, you must complete all levels and obtain a score of 3.

4. Technical Implementation

A part that I am proud of was having multiple constructors and learning about constructor hierarchy. By creating another constructor and calling it from the original allowed for the score to be incremented which wouldn't happen without the two constructors.

5. Most Challenging Part

The most challenging part of this project was encountering issues that dealt with things that I was unfamiliar with, such as the aforementioned constructor hierarchy issue. I overcame these issues by learning about new methods and techniques and applying them to my code.

6. Future Fixes and Updates

I think I could improve this game by adding some more interesting graphics because, in its current state, it looks pretty bleak and boring. I also noticed a few bugs that should be worked out (one such being if you press a colored square while the timer is ticking down, you would complete the level) and, if given the time, I would want to fix those.
